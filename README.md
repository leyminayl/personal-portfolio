# 🚀 Lei Min Aile Acera - Portfolio Website

Welcome to my personal portfolio website! This project is built as part of my **Web Analytics** subject at iACADEMY to showcase my skills in Web Development, UI/UX Design, and Data Tracking.

## 🔗 Live Demo
You can view the live site here: [https://lmnl-personal-portfolio.vercel.app/]

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Glassmorphism UI), JavaScript
- **Icons:** Font Awesome, DevIcon
- **Analytics:** Google Analytics 4 (GA4) with Event Tracking
- **Deployment:** Vercel / GitHub Pages

## 📊 Analytics Features (Web Analytics Focus)
For my Web Analytics subject, I integrated the following:
- **Global Site Tag (gtag.js):** To track overall page views and user demographics.
- **Custom Event Tracking:** I implemented a tracking script on the **"Download CV"** button to measure user engagement and conversion.
- **Real-time Monitoring:** Configured to monitor live traffic and user behavior patterns.

## 📁 Project Structure
- `index.html`: Main structure with GA4 integration.
- `style.css`: Custom styling with responsive design.
- `img/`: Assets including profile picture and downloadable CV.
